* Compressed files   
* Disc and media files

* Executable files   
* Font files  
* Image file formats   
* Internet related files   
* Presentation file formats   
* Programming files  
* Spreadsheet file formats   
* System related file formats   
* Video file formats   
* Word processor and text file formats  
* Torrents
